Simple Ajax Spotify now playing

A simple tool written in Jquery and PHP to show the now playing, or most recent listened to track on Spotify, using Last.FM!

What you need

  • An account with Last.fm
  • Last.FM API Key (get here)
  • A Spotify Account/Client
  • Minimal Knowledge of CSS/HTML
  • Webhosting with PHP 5.x

Setting up

Simply edit the last.fm.php and where you see

  1. Create an account, and get API key from LAST.fm
  2. Enable "Scrobble" to Last.fm from Spotify Client
  3. Configure PHP files, and add to HTML as listed bellow

There is only a few parts that'll require editing in the PHP file, these are your Last.FM username, and API key. Simply add this information inbetween the quotation variables as shown bellow. Once this is done, the next step is adding Jquery to load this file on page load. 
$user="yourusername";
$key="yourkey";

Here is an example script, you can use this one directly, or use your own custom one for more custom loading. You should also have some custom CSS, simply change the div 'content' to match your CSS. 
<script>
        $(document).ready(function(){
            $.ajax({
                type: 'POST',
                url: './last.fm.php',
                data: { request: 'true' },
                success: function(reply) {
                    $('.content').html(reply);
                }
            });
        });
    </script>

The last step will be to add the div and class, to tell the Jquery where to load the text. This is an example of what you can do, this depends on your websites styling. 

<div class="content"><center><div style="padding-top:120px;"><p><small>Loading...</small><br/><img src="./img/ajax/ajax-loader6.gif"/></p></div></center></div>

How it works

You need to setup your Spotify account to "scrobble" to your last.fm account. This will share your activity to Last.FM. Once you do this, when someone loads your page, a request is sent to the LastFM servers where it returns the latest played songs, the script will then determine if its now playing or recent.

Results

If you last listened to a track (not in private session) over a day ago:
I last listened to Strobe by deadmau5 over a day ago on Spotify
If you last listened to a track less than a day ago, but not current:
I last listened to Strobe by deadmau5 3 hours and 25 minutes on Spotify.
If you are currently listening to a track:
I am currently listening to Strobe by deadmau5 on Spotify.
